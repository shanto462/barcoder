Privacy Policy for Barcoder Chrome Extension

Last updated: October 6, 2024

Overview

Barcoder is a Chrome extension that generates barcodes from text selected by users on web pages. This privacy policy explains how we handle your data when you use our extension.

Data Collection and Usage

Text Selection: When you select text and use the "Generate Barcode" feature, the selected text is processed locally within your browser to generate a barcode. This text is not transmitted to any external servers or stored anywhere.

No Data Storage: Barcoder does not store, collect, or transmit any personal information, user data, or selected text to external servers or third parties.

Local Processing Only: All barcode generation happens entirely within your browser using the JsBarcode library. No data leaves your device.

Permissions

Barcoder requests the following permissions:

  • contextMenus: To add the "Generate Barcode" option to the right-click menu
  • activeTab: To access the currently active tab for text selection

These permissions are minimal and necessary for the extension's core functionality.

Third-Party Libraries

Barcoder uses the JsBarcode library (MIT License) for barcode generation. This library is included locally within the extension and does not transmit data to external servers.

Data Security

Since Barcoder processes all data locally within your browser, your selected text and generated barcodes remain private and secure on your device.

Updates to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last updated" date.

Contact

If you have any questions about this privacy policy, please contact us through the Chrome Web Store listing or create an issue on our GitHub repository.

Compliance

This extension complies with Chrome Web Store policies and does not collect, store, or transmit any user data to external servers.
